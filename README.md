# Salalah Eye — Brandbook

Official brand guidelines for **Salalah Eye** ([salalaheye.om](https://salalaheye.om)) — observation wheel & viewpoint, Salalah, Oman.

A single self-contained `index.html` (Tailwind + neomorphism UI) with:

- **Concept** of the abstract mark (eye + observation wheel + sea + sun + viewpoint)
- **Logo & wordmark** lockups — English `Salalah Eye` and Arabic `عين صلالة`
- **Construction**: grid, clear space, minimum size
- **Versions**: color, reversed, monochrome, on-gold
- **Usage rules** (do / don't)
- **Colors** with HEX · RGB · CMYK · Pantone · RAL
- **Typography**: Latin (Space Grotesk / Inter) + Arabic (Reem Kufi / Tajawal strict, Aref Ruqaa calligraphic), font colors and gradient accents
- **Moodboard** and **additional assets** (favicon, avatar, app icon, business cards, pattern)

## Features

- 🌗 Light / dark theme switcher (follows the system by default)
- 🌐 Trilingual switcher — **EN** (default) · RU · AR (with full RTL)
- ⬇️ Per-element **download** buttons (SVG + PNG) and a one-click **full kit (ZIP)**
- 📱 Mobile-first, responsive across all devices
- ✨ Motion via CSS + IntersectionObserver, enhanced with GSAP

## Run locally

Just open `index.html` in a browser — no build step. (Tailwind, fonts, GSAP and JSZip load from CDN, so an internet connection is required.)
